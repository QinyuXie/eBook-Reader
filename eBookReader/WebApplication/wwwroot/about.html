About the Website

  • This is a website with two sections: a public static section using HTML5, CSS3, and ES6, and a password protected section using Asp.Net Core MVC.
  • User has the ability to select and view both publicly available documents, and, with authentication, private documents.
  • There are controls to change the viewing scale and to iterate through pages or sections of any of the documents.
  • User has the ability to comment on documents for review. Comments are viewable only to the admin and the authenticated user that supplied the input. Comments don't have to be part of the document, but must be associated with a document being reviewed.
  • Admin has the ability to upload, replace, and delete any of the stored documents. Sources for the documents may be any file of a supported format from the local file system.
